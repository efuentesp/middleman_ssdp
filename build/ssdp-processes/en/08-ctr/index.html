Component Construction

It is the process of designing the internal structure (identifying and designing technical components), depth coding and testing a functional component

Inputs Steps Outputs

  1. Inspect the functional specification of the component
  2. Planning construction tasks of functional component
  3. Design internal structure of the component
  4. Design and prepare functional test
  5. Code component
  6. Compile code
  7. Execute functional test
  8. Evaluate deployment to integration environment
  9. Promote functional components to integration environment
  10. Post-mortem CTR.
  • FCI. Functional Components Implemented (for components to integrate).
    • Internal structure design
    • Source Code (one or more files)
    • Compilation Script (job or IDE project)
    • Deployment Script (job or IDE project)
    • Unit automated Testware (i.e JUnit o NUnit)
    • Functional Testware (script and data)
    • Unit test execution evidences and defect log
    • Self-generated documents (eg JavaDocs)
