Modelos de Compromiso

La oferta de Application Development se ha segmentado para poder ajustarse mejor a las necesidades y contextos de los clientes, para ello se han definido 6 modelos de compromiso:
  • Staffing
  • Managed Team
  • Baselined Project
  • Incremental Project
  • Component Factory
  • Project Portfolio Outsourcing
Cada modelo de compromiso tiene sus propias características que deben ser verificadas con el cliente, y tienen su propio modelo de operación.
En la parte inferior de esta página se tienen disponibles los audiovisuales que proporcionan un mayor detalle al respecto.
