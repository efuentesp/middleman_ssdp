Project Tracking and Oversight

Process for including appropriate control mechanisms for the advancement and progress of project performance metrics and decision making based on data and establish corrective action on deviations in order to ensure successful completion on time and budget.

Process Diagram


Process Description

Inputs Steps Outputs
  • Master Work Plan
  • PDP. Project Development Plan
  • PTL. Project Tracking Log:
  1. Project Development
  2. Metrics review project progress
  3. Project Advance Communication
  4. Scope Change Project documented
  5. Project Quality Products
  6. Participants assigned to the Project
  • MWP. Master Work Plan (updated)
  • PTL. Project Tracking Log updated:
    • RSK.Risk
    • ISS.Issues
    • PCS.Production Control
    • PCR.Project Change Request
    • TRM.Traceability Matrix
    • DEF.Defects
    • ACT.Action Items
    • MIN.Meeting Minute
    • BAKL.Backup Log
  • PPP. Project progress presentations

Roles

Name Description
PL. Project Leader He is responsible along with their team to set the strategy and planning of the project to meet customer commitments.
DM. Delivery Manager Responsible for the direction accountable for assigned projects.
TM. Team Members Responsible for update data on the project plan based on actual values for schedule duration and effort applied for assigned task and deliverables.
